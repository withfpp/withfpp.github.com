하루프레스 첫 게시글

하루프레스를 시작!

하루프레스로 블로깅을 시작한다. javascript 기반으로 정적 콘텐츠를 깃허브를 이용해 배포하는 점이 매우 매력적이다. 하루패드에 대해서는 오래전부터 들어왔지만, 우연히 하루프레스를 이용해 양질의 콘텐츠를 발행하는 블로그를 만나게 되었다.

차근차근 지식을 정리할 수 있도록, 잘 활용해 보도록 해야겠다.

땡큐

하루프레스를 개발자님께 존경을 표하고싶다. 정말 멋지다!
